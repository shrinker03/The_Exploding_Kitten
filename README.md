# Exploding Kitten Game

> A MERN project (MongoDB, Express.js, React, Node.js)

### Project Setup

```bash
# Create .env file
> touch .env

#add MONGODB_URI="YOUR_DB_CREDENTIALS" in .env file

```

## Build Setup

```bash

# Setup & Run server

yarn install
yarn server


# Run client
cd client/
yarn install
yarn start


```
